Adds a direct to the buffer.

Parameters
asm_structThe asm structure.
paramparam The param to check and add.
cmd_positionThe position of the command.
Returns
true if the add was successful, false otherwise.
Warning
Returns false if an error occures.
Note
Will determine if the param is a label or a number.
If it's a label, it will add a call to the label and add space for the address. (4 bytes)
If it's a number, it will add the number to the buffer.
