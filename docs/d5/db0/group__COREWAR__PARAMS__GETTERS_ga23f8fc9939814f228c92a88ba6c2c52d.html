Checks if the progname has a progname and a load address. Adds them to the params structure.

Parameters
argvThe arguments of the program.
paramsThe params structure.
progThe current prog.
mall_v1The value to multiply the size of the malloc. USE '1'.
Returns
true if the progname has a progname and a load address, false otherwise.
Note
Returns false in those cases:
- argv is NULL.
- params is NULL.
- array_len is inferior to 5.
- argv[0] is not "-n".
- argv[1] is not a number.
- argv[2] is not "-a".
- argv[3] is not a number.
- argv[4] is not a .cor file.
- prog is superior or equal to params->prog_nbr.
- my_strdup failed.
