Add a call to a label.

Parameters
asm_structThe asm struct.
label_nameThe label to add the call to.
callThe call to add.
Returns
true if the call was added, false otherwise.
Warning
Returns false if an error occurs.
Note
Returns false if the label is not found.
Returns false if an error occurs while allocating memory.
Returns true if the call was added.
A call is composed of :
- The position of the command that contains the call.
- The position of the call. (The position at which we'll write)
- The size of the call. (2 or 4)
