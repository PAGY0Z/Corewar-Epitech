asm_t * init_asm_struct ( char const *  filepath)

Initialize the asm structure.

Parameters
filepathThe path of the file to read.
Returns
The initialized asm structure.
Warning
The asm structure must be freed with free_asm_struct.
Note
The asm structure is initialized with :
- The file buffer. (empty)
- The file lines. (From the file)
- The new file name. (The file name with the new extension)
- The labels. (Empty from data except their names)
- The index. (initialized to 0)
- The line index. (initialized to 0)
