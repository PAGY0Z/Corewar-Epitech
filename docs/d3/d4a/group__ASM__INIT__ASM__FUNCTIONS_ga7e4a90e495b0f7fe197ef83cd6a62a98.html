labels_t ** convert_labels ( char **  labels_name)

Convert the labels names array to a labels_t structure array.

Parameters
labels_namesThe labels to convert.
Returns
The converted labels.
Warning
Returns NULL if an error occurs.
Note
Returns NULL if an error occurs while allocating memory.
Each label is initialized with :
- The label name.
- The label address (set to 0).
- The label command position (set to NULL).
- The label sizes (set to NULL).
- The label position (set to NULL).
- The label number of calls (set to 0).
The last element of the array is set to NULL.
