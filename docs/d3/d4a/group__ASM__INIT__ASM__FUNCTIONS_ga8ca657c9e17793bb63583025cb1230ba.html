int init_asm_programm ( char const *  filepath)

Init the asm programm execute it.

Parameters
filepathThe path of the file
Returns
0 if success, 84 if error
Note
The asm programm is initialized & ran with these steps:
- The asm structure init.
- The file parsing.
- The file writing.
- The asm structure freeing.
If an error occurs during the execution, the asm structure is freed.
The whole programm is stopped nicely.
