champ_t * champdup ( champ_t *  champ,
int  new_champ_pc,
int *  mall_vs 
)

Duplicates a champ structure.

Parameters
champA pointer to the champ structure.
new_champ_pcThe new pc of the champ.
mall_vsAn array of int containing the size of the mallocs.
Returns
A pointer to the new champ structure.
Note
Returns NULL in those cases:
- The malloc of the new champ structure failed.
- The my_strdup of the comment failed.
- The my_strdup of the name failed.
- The malloc of the params failed.
- The malloc of the regs failed.
Functions steps:
- Mallocs the new champ structure.
- Duplicates the comment.
- Duplicates the name.
- Mallocs the params.
- Mallocs the regs.
- Copies the params.
- Copies the regs.
- Sets the new pc.
- Sets the prog_nbr.
- Sets the cycle_to_wait to -1.
- Copies the carry.
- Copies the alive_during_last_cycle.
