bool champion_is_waiting ( champ_t *  champ)

Checks if a champion is waiting.

Parameters
champA pointer to the champion structure.
Returns
True if the champion is waiting, False otherwise.
Note
Returns true in those cases:
- champ is NULL
- champ->cycle_to_wait is more than 0
Returns false in those cases:
- champ->cycle_to_wait is less than or equal to 0
