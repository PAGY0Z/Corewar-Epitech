void execute_cycle ( corewar_t *  global)

Executes a cycle.

Parameters
globalA pointer to the global structure.
Returns
Nothing.
Note
Functions steps:
- For each champion:
- If the champion is dead (pc == -1), continue.
- If the champion is waiting, continue.
- If the instruction is valid, execute it.
- Update the champion parameters.
- If the instruction is invalid, reset the champion parameters.
- Update the champion program counter.
- Set the cycle to wait to -1.
- If the instruction is valid, set the cycle to wait to the
instruction cycle to wait.
