The parsing byte checker for or instruction.

Parameters
op_codeA byte which represent the instruction
pars_byteA byte which represent the parsing byte
Returns
True if the arguments are correct, False otherwise.
Note
Returns false in those cases:
- op_code is not 0x07
- bits 0b{11} 00 00 00 are not REG or DIR or IND
- bits 0b00 {11} 00 00 are not REG or DIR or IND
- bits 0b00 00 {11} 00 are not REG
- bits 0b00 00 00 {11} are not OTH
