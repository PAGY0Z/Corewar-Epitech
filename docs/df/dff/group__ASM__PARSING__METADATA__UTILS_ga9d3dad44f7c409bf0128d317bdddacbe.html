Check if a line is a .comment line.

Parameters
lineThe line to check.
Returns
true if the line is a .comment line, false otherwise.
Note
A .comment line is a line that follows the following format :
[SPACES] .comment SPACE "COMMENT" [SPACES]
All the checks done :
- The line is not NULL.
- The line is not empty.
