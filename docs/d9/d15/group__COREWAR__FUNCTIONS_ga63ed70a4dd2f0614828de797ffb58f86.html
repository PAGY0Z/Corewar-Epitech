Parameters

argvThe arguments.
Returns
true if the arguments are valid, false otherwise.
Note
Returns false if one of the arguments is invalid.
An argument is invalid if :
- It is not '-dump'
- It is not '-n'
- It is not '-a'
- It is not a number
