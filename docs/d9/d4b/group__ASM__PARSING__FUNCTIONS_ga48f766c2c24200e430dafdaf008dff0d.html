bool line_treatment ( asm_t *  asm_struct,
char *  line 
)

Treat a single line. Divide it into words and add the instruction to the buffer.

Parameters
asm_structThe asm struct.
lineThe line to treat.
Returns
true if the line was correctly treated, false otherwise.
Note
Return false if the asm_struct or the line is NULL.
Return false if the line could not be divided into words.
Case 1 : The first is a registered label.
- The label adress is updated.
- The instruction is added to the buffer. line_words + 1 is passed
Case 2 : The first word is not a registered label.
- The instruction is added to the buffer. line_words is passed
