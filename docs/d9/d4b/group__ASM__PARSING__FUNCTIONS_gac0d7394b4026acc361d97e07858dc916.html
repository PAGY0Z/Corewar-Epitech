◆ file_parsing()

bool file_parsing ( asm_t *  asm_struct)

Parse the file.

Parameters
asm_structThe asm struct.
Returns
true if the write was successful, false otherwise.
Note
The file will be parsed in this order:
- Add Magic number
- Add Name
- Add Comment
- Add The instructions
- Replace the program size by its value
- Replace the labels by their address
Returns false if the name or the comment is not found.
Returns false if an error occurs during the parsing.
8 bytes are added to the buffer after the name. 4 bytes as padding and 4 bytes for the program size.
4 bytes are added to the buffer after the comment. 4 bytes as padding.
