bool add_name ( asm_t *  asm_struct)

Add the name to the buffer.

Parameters
asm_structThe asm struct.
Returns
true if the write was successful, false otherwise.
Note
Functions steps:
- In a loop, go through the lines of the file.
- If the line is empty, go to the next line.
- If the line is not a name, return false.
- If the line is a name, add it to the buffer.
- Return true.
Return false if no name is found.
