bool add_comment ( asm_t *  asm_struct)

Add the comment to the buffer.

Parameters
asm_structThe asm struct.
Returns
true if the write was successful, false otherwise.
Note
Functions steps:
- In a loop, go through the lines of the file.
- If the line is empty, go to the next line.
- If the line is not a comment, return false.
- If the line is a comment, add it to the buffer.
- Return true.
Return false if no comment is found.
