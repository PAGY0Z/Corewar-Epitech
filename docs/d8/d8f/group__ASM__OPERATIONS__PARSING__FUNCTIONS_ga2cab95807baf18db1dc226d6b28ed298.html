bool parsing_05_sub ( asm_t *  asm_struct,
char **  line 
)

Check if the sub operation is well written and add it to the buffer.

Parameters
asm_structThe asm struct.
lineThe line to parse.
Returns
true if the line was parsed correctly, false otherwise.
Note
Return false in those cases:
- The line is not 4 words long.
- The first word is not "sub".
- The instruction byte could not be added to the buffer.
- The parsing byte could not be added to the buffer.
- The second word is not a register.
- The third word is not a register.
- The fourth word is not a register.
