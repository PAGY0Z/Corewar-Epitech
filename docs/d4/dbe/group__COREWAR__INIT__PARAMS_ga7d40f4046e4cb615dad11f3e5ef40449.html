Get the programs params.

Parameters
argvThe arguments.
paramsThe params structure.
mall_v1The value to multiply the malloc size by. USE '1'.
Returns
true if the programs params are valid, false otherwise.
Note
Returns false if an error occurs.
The progs params structure array is NULL terminated.
Algorithm :
- y is the index of the current prog.
- slider is the number of arguments to skip.
- if the current argument is a program name :
- add the program name to the progs params structure array.
- skip 1 argument.
- if the current argument is an adress & a program name :
- add the program name to the progs params structure array.
- add the adress to the progs params structure array.
- skip 3 arguments.
- if the current argument is a program number & a program name :
- add the program name to the progs params structure array.
- add the program number to the progs params structure array.
- skip 3 arguments.
- if the current argument is an adress, a program number & a program name :
- add the program name to the progs params structure array.
- add the program number to the progs params structure array.
- add the adress to the progs params structure array.
- skip 5 arguments.
- if the current argument is a program number, an adress & a program name :
- add the program name to the progs params structure array.
- add the program number to the progs params structure array.
- add the adress to the progs params structure array.
- skip 5 arguments.
- if the current argument is none of the above :
- return false.
