Initialize the params structure.

Parameters
argvThe arguments.
mall_v1The malloc value for the first malloc. USE '1'.
mall_v2The malloc value for the second malloc. USE '1'.
Returns
The params structure.
Note
Returns NULL if an error occurs.
Returns the params structure if no error occurs.
Initializes the params in this order:
- 1. The dump_nbr_cycles.
- 2. Get the number of programs.
- 3. Initialize the params_progs_t structure array.
- 4. Get the programs parameters.
